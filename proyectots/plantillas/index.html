¿Qués es la Simulación?

Es el proceso de diseñar y desarrollar un modelo computarizado de un sistema o proceso y conducir experimentos con este modelo con el propósito de entender el comportamiento del sistema o evaluar varias estrategias con las cuales se puede operar un sistema, según Shanon Robert.

¿Por qué es importante?

Da el soporte necesario al diseñador durante el proceso de diseño, análisis y diagnosis de sistemas. El software debe complementar el talento del diseñador para que éste pueda modelar y simular de forma lo mas eficientemente posible.
